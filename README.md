# DIO-Imposto-Renda
Tabela para Preenchimento de informações sobre IRPF

Este repositório tem como objetivo aplicar os conceitos de Excel no desenvolvimento de uma ferramenta prática de tabela para preenchimento de informações relacionadas ao IRPF.

Material desenvolvido para fins didáticos, a planilha não contém todas as entradas necessárias para uma declaração completa.

# Conteúdo
- Planilha em formato .xlsx
- Planilha somente para preenchimento de informações, sem nenhum cálculo executado.


# Observação
- O objetivo é que o o usuário preencha a planilha e envie a algum profissional para fazer o lançamento dos dados no programa da Receita Federal.
- As abas são separadas conforme o aplicativo da Receita Federal.
